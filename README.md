A Wi-Fi synchronized Arc Reactor-inspired digital clock built with an ESP32, a TM1637 4-digit display, and a NeoPixel LED ring.

This project combines a sci-fi-inspired 3D printed design with real-time clock functionality. The ESP32 connects to an NTP server to keep accurate time, automatically adjusts for daylight saving time, and displays the current time on a TM1637 display while illuminating the reactor with customizable RGB lighting.

Features

* 🌐 Automatic Wi-Fi time synchronization using NTP
* 🕒 Accurate digital clock with TM1637 4-digit display
* 💡 Addressable NeoPixel LED ring with adjustable brightness
* ⚡ Hourly “Arc Reactor” flash animation
* 🖨️ Fully 3D printed enclosure inspired by Iron Man’s Arc Reactor
* 🔧 Easily customizable colors, brightness, and time zone

Hardware

* ESP32
* TM1637 4-digit display
* WS2812B (NeoPixel) LED ring
* 3D printed Arc Reactor housing
* Power supply (5V)

Software

* Arduino IDE
* Adafruit NeoPixel
* TM1637Display
* NTPClient
* WiFi
* WiFiManager
